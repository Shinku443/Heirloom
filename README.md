# Command List

## User Commands

By default all user commands are prefixed with the `/` character.  

e.g. To use the bot's map command, enter `/map`

| Command    | Description |
|------------|-------------|
| craft      | Get the current replicator items                                                        |
| create     | Creates a matchmaking lobby to let users in your server know what game you want to play |
| map        | Get the current and next map                                                            |
| patch      | Get the latest patch notes                                                              |
| status     | Checks server status                                                                    |
| store      | Get current in game shop bundles                                                        |
